## 😵 레이어 100개 도면에서 특정 레이어만 보고 싶을 때

상상해보세요.

**레이어가 100개 넘는 도면**을 받았어요.

- 건축 레이어 50개
- 구조 레이어 30개
- 설비 레이어 20개
- 기타 레이어들...

근데 지금 당장 **"전기-콘센트" 레이어만** 확인하고 싶다고요.

보통은 어떻게 하나요?

```
1. 레이어 특성 관리자 열기 (LA)
2. 스크롤 내리면서 하나씩 전구 아이콘 클릭 (99번...)
3. "전기-콘센트" 레이어만 ON
4. 확인 버튼 클릭
```

**손목 터짐 예정!** 😱

레이어가 많으면 많을수록 지옥이죠.

그래서 고민했어요.

> "클릭 한 번에 내가 원하는 레이어만 딱 보면 안 될까?"

그 고민의 결과물이 바로 **LOOO 리습**입니다! 🎯

---

## ⚡ LOOO 리습이 뭘 하는 건데요?

딱 하나만 합니다.

**선택한 레이어만 켜고, 나머지 전부 끔!**

그게 다예요. 심플하죠? 😎

---

## 📖 사용 방법 (2가지 방식)

### 방법 1: 레이어 이름 직접 입력

```
명령: LOOO ↵
레이어 이름을 입력하거나 Enter를 눌러 객체를 선택하십시오: 전기-콘센트 ↵
선택된 레이어: 전기-콘센트
작업이 완료되었습니다.
```

**끝!** 이제 "전기-콘센트" 레이어만 보입니다! ✨

### 방법 2: 객체 클릭해서 선택 (더 쉬움!)

```
명령: LOOO ↵
레이어 이름을 입력하거나 Enter를 눌러 객체를 선택하십시오: ↵ (Enter만 누름)
레이어 정의된 객체를 선택하십시오: [콘센트 심볼 클릭!]
선택된 레이어: 전기-콘센트
작업이 완료되었습니다.
```

**레이어 이름을 몰라도 됩니다!**

그냥 보고 싶은 객체 클릭하면 끝! 👍

## 🎯 실전 활용 예시

### 예시 1: 전기 설비만 확인

**상황:** 협력업체에서 받은 통합 도면에서 전기 콘센트 위치만 확인하고 싶음

```
명령: LOOO
[아무 콘센트 심볼 클릭]
```

**결과:**
- ✅ 전기-콘센트 레이어 → ON
- ❌ 나머지 99개 레이어 → OFF

**화면이 갑자기 깨끗해짐!** 🎉

---

### 예시 2: 구조 벽체만 추출

**상황:** 건축 도면에서 구조 벽체만 따로 보고 싶음

```
명령: LOOO
레이어 이름: 구조-벽체
```

**결과:**
- 가구, 문, 창문, 치수선... 다 사라짐
- 순수 구조 벽체만 딱!

**구조 검토할 때 완전 꿀!** 🍯

---

### 예시 3: 복잡한 배관도에서 특정 계통만

**상황:** 급수, 배수, 소화, 난방... 배관이 엉켜있는데 급수만 보고 싶음

```
명령: LOOO
[급수 배관 클릭]
```

**결과:**
- 화면이 심플해짐
- 급수 계통만 선명하게 보임
- 간섭 검토 완전 쉬워짐! ✨

## 💡 꿀팁 모음

### 팁 1: 현재 레이어는 자동으로 켜짐

```
현재 작업 레이어: 치수선
선택한 레이어: 전기-콘센트
```

이 경우 **두 레이어 모두 ON** 상태가 됩니다!

왜냐하면 현재 레이어가 꺼지면 **작업을 못 하니까요**. 똑똑하죠? 🧠

### 팁 2: Undo 그룹으로 한 방에 되돌리기

작업 후에 원래 상태로 돌아가고 싶다면?

```
명령: U ↵
```

**한 번만 누르면 모든 레이어가 원래 상태로!**

Undo 그룹 기능이 내장되어 있어요. 👍

### 팁 3: 존재하지 않는 레이어 입력하면?

```
레이어 이름: 없는레이어
지정한 레이어 정의되지 않습니다.
```

친절하게 에러 메시지 나오고 작업 취소됩니다. 안전함! 🛡️

### 팁 4: 여러 레이어 반복 확인할 때

```
명령: LOOO → 전기 확인
명령: U → 되돌리기
명령: LOOO → 배관 확인
명령: U → 되돌리기
```

**U(Undo)와 LOOO를 번갈아 쓰면** 여러 레이어를 빠르게 확인 가능! 🚀

## 🔥 이 리습의 장점

✅ **초고속** - 클릭 한 번에 정리 완료
✅ **직관적** - 객체만 클릭하면 됨
✅ **안전함** - Undo로 한 방에 복구
✅ **똑똑함** - 현재 레이어 자동 보호
✅ **에러 방지** - 잘못된 입력 시 친절한 안내

## 🛠️ 기술적 팁 (개발자용)

이 리습의 핵심 로직:

```lisp
;; 모든 레이어 끄기
(vl-cmdf "._LAYER" "_OFF" "*" "_N" "")

;; 선택한 레이어만 켜기
(vl-cmdf "._LAYER" "_ON" layer_name "")

;; 현재 레이어도 보호
(if (not (equal layer_name old_clayer))
  (vl-cmdf "._LAYER" "_ON" old_clayer "")
)
```

**vl-cmdf** 함수로 자동화, **Undo 그룹**으로 안전성 확보!
