## 목적

AutoCAD에서 특정 레이어만 확인하고 싶을 때, 레이어 특성 관리자에서 수십 개의 레이어를 일일이 끄는 작업은 비효율적입니다. LOOO 리습은 선택한 레이어만 표시하고 나머지를 자동으로 숨겨 작업 효율을 높입니다.

## 개요

LOOO(Layer Only) 리습의 주요 기능은 다음과 같습니다.

**주요 기능**
- 선택한 레이어만 표시
- 나머지 레이어 자동 숨김
- 레이어 이름 직접 입력 방식
- 객체 선택을 통한 레이어 지정 방식
- 현재 레이어 자동 보호

**기술적 특징**
- Undo 그룹 적용으로 한 번에 복구 가능
- Visual LISP 기반
- 빠른 실행 속도

## 다운로드

**파일명:** AN_LOOO_레이어단독표시.lsp

[다운로드 링크](https://raw.githubusercontent.com/cattowel39200/Error_404/main/AN_lisp/AN_LOOO_%EB%A0%88%EC%9D%B4%EC%96%B4%EB%8B%A8%EB%8F%85%ED%91%9C%EC%8B%9C.lsp)

**설치 방법**
1. 링크에서 우클릭하여 "다른 이름으로 저장"
2. 파일을 .lsp 확장자로 저장
3. AutoCAD에서 APPLOAD 명령 실행
4. 저장한 파일 선택 후 로드

## 사용 방법

### 방법 1: 레이어 이름 직접 입력

```
명령: LOOO
레이어 이름을 입력하거나 Enter를 눌러 객체를 선택하십시오: 전기-콘센트
선택된 레이어: 전기-콘센트
작업이 완료되었습니다.
```

### 방법 2: 객체 선택을 통한 레이어 지정

```
명령: LOOO
레이어 이름을 입력하거나 Enter를 눌러 객체를 선택하십시오: [Enter]
레이어 정의된 객체를 선택하십시오: [객체 클릭]
선택된 레이어: 전기-콘센트
작업이 완료되었습니다.
```

### 현재 레이어 보호 기능

현재 작업 중인 레이어는 자동으로 ON 상태를 유지합니다. 이는 작업 연속성을 보장하기 위한 기능입니다.

## 활용 예시

### 예시 1: 전기 설비만 확인

```
상황: 협력업체 통합 도면에서 전기 콘센트만 확인
명령: LOOO
객체: [콘센트 심볼 클릭]
결과: 전기-콘센트 레이어만 표시
```

전기 설비 위치를 빠르게 확인할 수 있습니다.

### 예시 2: 구조 벽체만 추출

```
상황: 건축 도면에서 구조 벽체만 확인
명령: LOOO
레이어 이름: 구조-벽체
결과: 구조 벽체만 표시
```

구조 검토 시 불필요한 요소를 제거하고 집중할 수 있습니다.

### 예시 3: 배관 계통 확인

```
상황: 복잡한 배관도에서 급수 계통만 확인
명령: LOOO
객체: [급수 배관 클릭]
결과: 급수 계통만 표시
```

간섭 검토나 수량 산출 시 유용합니다.

### 예시 4: 여러 레이어 반복 확인

```
명령: LOOO → 전기 확인
명령: U → 원상복구
명령: LOOO → 배관 확인
명령: U → 원상복구
```

Undo와 조합하여 여러 레이어를 순차적으로 확인할 수 있습니다.

## 주의사항

**존재하지 않는 레이어**
- 지정한 레이어가 없는 경우 오류 메시지가 표시됩니다.
- 작업이 취소되며 원래 상태가 유지됩니다.

**현재 레이어**
- 현재 작업 레이어는 항상 ON 상태로 유지됩니다.
- 선택한 레이어와 현재 레이어가 다른 경우 두 레이어 모두 표시됩니다.

**작업 복구**
- U(Undo) 명령으로 모든 레이어를 원래 상태로 복구할 수 있습니다.
- Undo 그룹이 적용되어 있어 한 번의 명령으로 복구 가능합니다.

**AutoCAD 버전**
- AutoCAD 2000 이상, Visual LISP 필요
