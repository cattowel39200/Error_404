## 😫 같은 내용 10번 복붙하다가 미쳐버릴 뻔했어요

도면 작업하다 보면 이런 상황 많으시죠?

**상황:** 테이블에서 "콘크리트" 라는 텍스트를 10군데에 입력해야 함

**기존 방법:**
```
1. "콘크리트" 텍스트 더블클릭
2. 전체 선택 (Ctrl+A)
3. 복사 (Ctrl+C)
4. 첫 번째 빈칸 더블클릭
5. 붙여넣기 (Ctrl+V)
6. 두 번째 빈칸 더블클릭
7. 붙여넣기 (Ctrl+V)
8. 세 번째 빈칸 더블클릭...
```

**10번 반복!** 😱

손목이 아프고, 시간도 오래 걸리고, **짜증 폭발!** 💢

그래서 생각했습니다.

> "한 번 복사하면 클릭만으로 연속 붙여넣기되면 안 될까?"

그 결과물이 바로 **TTR 리습**입니다! 🎯

---

## ⚡ TTR 리습이 하는 마법

엄청 간단해요!

1. **원본 텍스트 클릭** (한 번만!)
2. **대상 텍스트들 클릭 클릭 클릭...** (연속으로!)
3. **Enter로 종료**

**도장 찍듯이** 텍스트가 쭉쭉 복사됩니다! ✨

---

## 📖 사용 방법 (클릭의 향연)

### 1단계: LISP 로드

`APPLOAD` → `AN_TTR_글자연속교체.lsp` 로드

### 2단계: 명령어 실행

```
명령: TTR
```

또는

```
명령: TR
```

(둘 다 같은 기능!)

### 3단계: 원본 선택

```
복사할 텍스트 객체를 선택하세요: [원본 클릭!]
원본 텍스트: "콘크리트"
```

### 4단계: 대상들 연속 클릭

```
이 내용을 적용할 텍스트 객체들을 선택하세요 (종료하려면 Enter):
[첫 번째 텍스트 클릭] → 텍스트 내용이 교체되었습니다.
[두 번째 텍스트 클릭] → 텍스트 내용이 교체되었습니다.
[세 번째 텍스트 클릭] → 텍스트 내용이 교체되었습니다.
...
[Enter 누르면 종료]
```

**끝!** 클릭만 하면 돼요! 🖱️

## 🎯 실전 활용 예시

### 예시 1: 테이블 일괄 수정

**Before:**

| 구분 | 재질 | 규격 |
|------|------|------|
| A형 | 철근 | 200 |
| B형 | 철근 | 250 |
| C형 | 철근 | 300 |

**철근을 모두 "콘크리트"로 변경하고 싶다면?**

```
명령: TTR
[철근 아무거나 하나 클릭] → "철근" 확인
[남은 철근들 클릭 클릭 클릭...]
Enter!
```

**After:**

| 구분 | 재질 | 규격 |
|------|------|------|
| A형 | 콘크리트 | 200 |
| B형 | 콘크리트 | 250 |
| C형 | 콘크리트 | 300 |

**5초 완료!** ⚡

---

### 예시 2: 범례 통일

**도면 여러 곳에 흩어진 범례 텍스트 통일:**

```
기존: "콘크리이트" (오타!)
     "concrete" (영문)
     "콘크릿" (오타!)

TTR로 "콘크리트" 하나 선택 후:
→ 클릭 클릭 클릭...

결과: 모두 "콘크리트"로 통일!
```

**오타 수정이 이렇게 쉬워요!** 👍

---

### 예시 3: 치수 주석 일괄 변경

**20개 치수선의 주석을 모두 "검토 중"으로:**

```
명령: TTR
["검토 중" 텍스트 선택]
[나머지 19개 텍스트 클릭 클릭 클릭...]
```

**1분 안에 완료!** ⚡

## 💡 꿀팁 모음

### 팁 1: TEXT와 MTEXT 모두 지원

단일행 텍스트든, 여러행 텍스트든 상관없어요!

```
원본: TEXT
대상: MTEXT → OK!

원본: MTEXT
대상: TEXT → OK!
```

**섞여 있어도 문제없음!** 👌

### 팁 2: 실수로 잘못 클릭하면?

```
[잘못된 객체 클릭]
"선택한 객체가 텍스트 객체가 아닙니다."
```

친절한 에러 메시지가 나오고, **작업은 계속 진행**돼요!

### 팁 3: 몇 개든 상관없어요

10개? 50개? 100개?

**Enter 누를 때까지 무한 반복!** ♾️

### 팁 4: 명령어 2개 (TTR, TR)

`TTR` (Text To Replace) 또는 `TR` (Text Replace)

편한 거 쓰세요! 😊

## 🔥 이 리습의 진짜 장점

✅ **무한 반복** - Enter 전까지 계속 가능
✅ **초고속** - 클릭만 하면 끝
✅ **직관적** - 설명 안 봐도 사용 가능
✅ **안전함** - 잘못 클릭해도 건너뛰기
✅ **호환성** - TEXT, MTEXT 모두 OK

## 🚀 실무 활용 시나리오

### 시나리오 1: 설계 변경

**상황:** 설계 변경으로 "H빔"을 모두 "I빔"으로 변경

**기존 방법:**
1. 찾기/바꾸기 (FIND) → 레이어, 폰트 등 설정 복잡
2. 또는 하나씩 수정 → 시간 많이 걸림

**TTR 방법:**
```
명령: TTR
["I빔" 텍스트 선택]
["H빔" 텍스트들 클릭 클릭...]
Enter!
```

**완료! 간단하죠?** ✨

---

### 시나리오 2: 도면 검토

**상황:** 도면 검토 중 여러 곳에 "검토 완료" 표시

```
명령: TTR
["검토 완료" 기존 텍스트 선택]
[빈 텍스트들 클릭 클릭...]
```

**도장 찍듯이 완료 표시!** 🎯
