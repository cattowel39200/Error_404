## 목적

AutoCAD 작업 중 동일한 텍스트 내용을 여러 위치에 반복 적용해야 할 때, 매번 복사-붙여넣기를 반복하는 것은 비효율적입니다. TTR 리습은 원본 텍스트를 한 번 선택한 후, 대상 텍스트들을 연속으로 클릭하여 내용을 일괄 교체합니다.

## 개요

TTR(Text To Replace) 리습의 주요 기능은 다음과 같습니다.

**주요 기능**
- 원본 텍스트 한 번 선택
- 대상 텍스트 연속 클릭으로 내용 교체
- TEXT 및 MTEXT 지원
- 교차 유형 교체 가능
- Enter로 작업 종료
- 명령어 2개 제공 (TTR, TR)

**기술적 특징**
- 무한 루프 구조로 연속 처리
- 텍스트 위치 및 속성 유지
- 빠른 실행 속도

1. 링크에서 우클릭하여 "다른 이름으로 저장"
2. 파일을 .lsp 확장자로 저장
3. AutoCAD에서 APPLOAD 명령 실행
4. 저장한 파일 선택 후 로드

## 사용 방법

### 기본 사용법

```
명령: TTR (또는 TR)
복사할 텍스트 객체를 선택하세요: [원본 클릭]
원본 텍스트: "콘크리트"
이 내용을 적용할 텍스트 객체들을 선택하세요 (종료하려면 Enter):
[대상1 클릭] → 텍스트 내용이 교체되었습니다.
[대상2 클릭] → 텍스트 내용이 교체되었습니다.
[대상3 클릭] → 텍스트 내용이 교체되었습니다.
[Enter] → 작업 종료
```

### 지원 객체

- TEXT (단일행 텍스트)
- MTEXT (여러행 텍스트)
- 원본과 대상의 유형이 달라도 교체 가능

## 활용 예시

### 예시 1: 테이블 일괄 수정

```
상황: 테이블에서 "철근"을 모두 "콘크리트"로 변경

명령: TTR
원본: [콘크리트 텍스트 클릭]
대상: [철근 텍스트들을 순차적으로 클릭]
Enter로 종료
```

테이블 데이터를 빠르게 일괄 수정할 수 있습니다.

### 예시 2: 범례 통일

```
상황: 도면 내 오타 및 표기 불일치 수정

기존:
- "콘크리이트" (오타)
- "concrete" (영문)
- "콘크릿" (오타)

작업:
TTR 명령으로 정확한 "콘크리트" 적용
```

범례 및 주석의 일관성을 유지할 수 있습니다.

### 예시 3: 치수 주석 일괄 변경

```
상황: 20개 치수선의 주석을 "검토 중"으로 일괄 변경

명령: TTR
원본: ["검토 중" 텍스트 선택]
대상: [19개 텍스트 순차 클릭]
```

반복 작업을 대폭 줄일 수 있습니다.

### 예시 4: 설계 변경 사항 반영

```
상황: "H빔"을 모두 "I빔"으로 변경

명령: TTR
원본: ["I빔" 선택]
대상: ["H빔" 텍스트들 클릭]
```

설계 변경 시 신속하게 대응할 수 있습니다.

## 주의사항

**객체 선택**
- 텍스트가 아닌 객체를 선택하면 오류 메시지가 표시됩니다.
- 작업은 계속 진행되며, 다음 객체를 선택할 수 있습니다.

**작업 종료**
- Enter 키를 눌러 작업을 종료합니다.
- 중간에 중단하고 싶을 때 언제든지 Enter를 누를 수 있습니다.

**연속 처리**
- Enter를 누르기 전까지 무제한으로 대상을 선택할 수 있습니다.

**AutoCAD 버전**
- AutoCAD 2000 이상에서 사용 가능합니다.
