## 😫 수량 산출할 때마다 블록인지 선인지 헷갈려요

도면에서 수량 산출하려고 하면 이런 고민 하시죠?

**상황:** 어린이보호구역 시설물 수량 산출

```
- 통합표지: 블록으로 그려져 있음 → 개수 세기
- 노면표시: 선/폴리선으로 그려져 있음 → 면적 계산
- 차선: 선으로 그려져 있음 → 길이 계산
```

**문제는...**

1. 뭐가 블록이고 뭐가 선인지 일일이 확인
2. 블록은 개수 세는 명령어 (BNTT)
3. 선은 수량 계산하는 명령어 (LLTT)
4. 두 개 따로 실행해서 따로 정리
5. 엑셀에 다시 통합...

**너무 복잡해!** 😱

그래서 생각했습니다.

> "자동으로 판별해서 한 번에 통계표 만들 수 없을까?"

그 답이 **GGGG 리습**입니다! 🎯

---

## ⚡ GGGG 리습의 똑똑함

**일반 명령어:**
```
블록 → BNTT 실행
선 → LLTT 실행
둘 다 있으면? → 두 번 실행 후 수동 통합
```

**GGGG 리습:**
```
뭐든지 선택 → 자동 판별 → 통계표 완성!
```

**AI처럼 자동 판별!** 🤖

---

## 📖 사용 방법 (선택만 하면 끝)

### 1단계: LISP 로드

`APPLOAD` → `AN_GGGG_어린이보호구역통계.lsp` 로드

### 2단계: 명령어 실행 및 선택

```
명령: GGGG
통계를 작성할 객체들을 선택하세요: [전체 선택!]
```

### 3단계: 자동 판별 및 통계 생성

```
50개의 객체가 선택되었습니다.

→ 블록과 선/해치가 혼합되어 있습니다.
→ 통합 통계표를 생성합니다.

[통계표가 자동으로 도면에 생성됨!]
```

**끝!** 알아서 판별하고 알아서 통계표 생성! 🎉

---

## 🎯 3가지 모드 자동 선택

### 모드 1: 블록만 있는 경우

```
선택: 통합표지 20개 (블록)

→ 블록 객체가 감지되었습니다.
→ 블록 통계를 생성합니다.

결과:
┌────────────────┬──────┐
│ 시설물 명칭    │ 수량 │
├────────────────┼──────┤
│ 통합표지(시점) │  5개 │
│ 통합표지(종점) │ 15개 │
└────────────────┴──────┘
```

**블록 개수 자동 집계!** ✅

---

### 모드 2: 선/해치만 있는 경우

```
선택: 노면표시 선 30개

→ 선/폴리선/해치 객체가 감지되었습니다.
→ 수량 통계를 생성합니다.

결과:
┌──────────────┬────────┬──────┐
│ 레이어       │ 길이   │ 면적 │
├──────────────┼────────┼──────┤
│ 노면표시-536 │ 120.5m │ 25㎡ │
│ 노면표시-537 │  85.3m │ 18㎡ │
└──────────────┴────────┴──────┘
```

**길이/면적 자동 계산!** ✅

---

### 모드 3: 블록 + 선/해치 혼합

```
선택: 통합표지(블록) + 노면표시(선) 혼합

→ 블록과 선/해치가 혼합되어 있습니다.
→ 통합 통계표를 생성합니다.

결과: 하나의 표에 블록과 선 통계 모두 포함!
┌────────────────┬──────┬────────┬──────┐
│ 항목           │ 개수 │ 길이   │ 면적 │
├────────────────┼──────┼────────┼──────┤
│ 통합표지(시점) │  5개 │    -   │  -   │
│ 노면표시-536   │   -  │ 120.5m │ 25㎡ │
└────────────────┴──────┴────────┴──────┘
```

**통합 통계표 자동 생성!** ✨

## 🎯 특화 기능: 어린이보호구역 시설물

이 리습은 **어린이보호구역 시설물**에 특화되어 있어요!

**62개 시설물 분류 체계 내장:**
```
- 통합표지(시점)
- 통합표지(종점)_해제
- 통합표지(종점)_속도
- 통합표지(세로형)
- 노면표시(536, 536-2, 536-3)
- 노면표시(536-4, 536-5)
...
총 62개 항목!
```

**블록 이름만 맞으면 자동으로 분류!** 🎯


**자동 검수!** 🔍

이제 **9개의 LISP 리습 시리즈**가 모두 완성되었습니다!

1. **BRN** - 블록 이름 변경
2. **TTC** - 텍스트 교환
3. **LOOO** - 레이어 단독 표시
4. **COOO** - 숫자 추출 복사
5. **TTR** - 텍스트 연속 복사
6. **JJJ** - 폴리라인 합치기 (Z좌표 보존)
7. **BLDEL** - 해치 일괄 삭제
8. **BCENT** - 블록 기준점 중심 변경
9. **GGGG** - 통계표 자동 생성
