## 목적

어린이보호구역 설계 도면에서 시설물 수량을 산출할 때, 블록은 개수를, 선과 해치는 길이와 면적을 별도로 계산해야 합니다. GGGG 리습은 선택한 객체가 블록인지 선인지 자동으로 판별하여 통합 통계표를 생성합니다.

## 개요

GGGG 리습의 주요 기능은 다음과 같습니다.

**주요 기능**
- 블록과 선 자동 판별
- 블록 개수 자동 집계
- 선 길이 및 면적 계산
- 통합 통계표 자동 생성
- 어린이보호구역 시설물 62종 분류 체계 내장

**기술적 특징**
- 3가지 자동 모드 (블록 전용, 선 전용, 혼합)
- 레이어 기반 분류
- 도면 내 자동 배치
- Visual LISP 기반

**지원 객체**
- 블록 (INSERT)
- 선 (LINE, LWPOLYLINE, POLYLINE)
- 해치 (HATCH)

## 사용 방법

### 기본 사용법

```
명령: GGGG
통계를 작성할 객체들을 선택하세요: [전체 선택]
50개의 객체가 선택되었습니다.

→ 블록과 선/해치가 혼합되어 있습니다.
→ 통합 통계표를 생성합니다.

[통계표가 자동으로 도면에 생성됨]
```

### 3가지 자동 모드

**모드 1: 블록만 선택**
```
선택: 통합표지 20개 (블록)

→ 블록 객체가 감지되었습니다.
→ 블록 통계를 생성합니다.

결과:
┌────────────────┬──────┐
│ 시설물 명칭    │ 수량 │
├────────────────┼──────┤
│ 통합표지(시점) │  5개 │
│ 통합표지(종점) │ 15개 │
└────────────────┴──────┘
```

**모드 2: 선/해치만 선택**
```
선택: 노면표시 선 30개

→ 선/폴리선/해치 객체가 감지되었습니다.
→ 수량 통계를 생성합니다.

결과:
┌──────────────┬────────┬──────┐
│ 레이어       │ 길이   │ 면적 │
├──────────────┼────────┼──────┤
│ 노면표시-536 │ 120.5m │ 25㎡ │
│ 노면표시-537 │  85.3m │ 18㎡ │
└──────────────┴────────┴──────┘
```

**모드 3: 혼합 선택**
```
선택: 통합표지(블록) + 노면표시(선) 혼합

→ 블록과 선/해치가 혼합되어 있습니다.
→ 통합 통계표를 생성합니다.

결과: 하나의 표에 블록과 선 통계 모두 포함
┌────────────────┬──────┬────────┬──────┐
│ 항목           │ 개수 │ 길이   │ 면적 │
├────────────────┼──────┼────────┼──────┤
│ 통합표지(시점) │  5개 │    -   │  -   │
│ 노면표시-536   │   -  │ 120.5m │ 25㎡ │
└────────────────┴──────┴────────┴──────┘
```

## 활용 예시

### 예시 1: 어린이보호구역 설계

```
상황: 스쿨존 시설물 수량 산출

1. 도면에서 전체 선택 (Ctrl+A)
2. GGGG 실행
3. 통계표 자동 생성

결과:
- 통합표지 15개
- 노면표시 120.5m
- 기타 시설물...

통계표를 납품 서류에 첨부
작업 시간: 5초
```

어린이보호구역 설계 서류 작성이 신속하게 완료됩니다.

### 예시 2: 설계 변경 후 수량 비교

```
상황: 설계 변경 후 수량 확인

Before:
GGGG 실행 → 통합표지 10개

[설계 변경]

After:
GGGG 다시 실행 → 통합표지 15개

차이: +5개 확인
```

변경 수량을 즉시 파악할 수 있습니다.

### 예시 3: 협력업체 도면 검수

```
상황: 받은 도면의 수량표 정확성 확인

도면 수량표: "통합표지 20개"
GGGG 실행 → 실제: 18개

오류 발견: 2개 부족
```

자동 검수로 오류를 빠르게 발견할 수 있습니다.

## 주의사항

**블록 이름 규칙**
- GGGG는 블록 이름으로 시설물을 분류합니다.
- 예: "011" → "통합표지(시점)"
- 블록 이름이 표준과 다르면 인식되지 않습니다.

**62개 분류 체계**
- 리습에 내장된 62개 항목만 자동 분류됩니다.
- 다른 시설물은 별도 커스터마이징이 필요합니다.

**레이어 정리**
- 레이어가 정리되어 있으면 통계표도 깔끔하게 생성됩니다.

**AutoCAD 버전**
- AutoCAD 2000 이상에서 사용 가능합니다.
