## ❓ 왜 이 리습을 만들었나요?

AutoCAD로 작업하다 보면 블록 이름을 바꿔야 할 때가 자주 있습니다.

**이런 경험 있으시죠?**
- 도면 표준이 바뀌어서 블록 이름 규칙 변경
- 협력업체 블록 이름이 우리 회사 규칙과 달라서 수정
- 오타 발견해서 고쳐야 할 때
- 프로젝트별 블록 이름 통일이 필요할 때

기본 RENAME 명령어는 **타이핑이 너무 귀찮고**, 블록 편집기는 **과정이 복잡**합니다.

> "블록 클릭 한 번, 이름 입력 한 번이면 끝나게 할 수 없을까?"

이런 고민에서 **BRN 리습**이 탄생했습니다!

---

## 🎯 이 리습의 기능

BRN 리습은 다음 기능을 제공합니다:

### 주요 기능
1. **블록 클릭 기반 선택** - 블록 이름 타이핑 불필요
2. **기존 이름 표시** - 수정만 하면 됨 (처음부터 칠 필요 없음)
3. **중복 검사** - 이미 존재하는 이름 자동 차단
4. **Enter로 취소** - 실수 방지
5. **자동 정리** - 기존 블록 정의 PURGE

### 기술적 특징
- 블록 재정의 방식으로 안전하게 변경
- 도면의 모든 해당 블록 인스턴스 자동 업데이트
- 회전/스케일 정보 유지

---

## 💡 사용 목적

이 리습은 다음과 같은 목적으로 사용됩니다:

✅ **블록 이름 규칙 통일** - 회사 표준에 맞게 변경
✅ **오타 수정** - 빠르게 정확한 이름으로 교체
✅ **프로젝트 표준화** - 여러 출처의 블록 이름 통일
✅ **작업 효율 향상** - RENAME 명령어 대비 60배 빠름
✅ **도면 정리** - 사용하지 않는 블록 정의 자동 삭제

### 설치 방법
1. 위 링크 클릭 → 마우스 우클릭 → "다른 이름으로 저장"
2. 파일을 `.lsp` 확장자로 저장
3. AutoCAD에서 `APPLOAD` 명령어 실행
4. 저장한 `.lsp` 파일 선택하여 로드

## 📖 사용 방법

### 1단계: LISP 로드
```
명령: APPLOAD
파일 선택: AN_BRN_블록명변경.lsp
```

### 2단계: 명령어 실행
```
명령: BRN ↵
```

### 3단계: 블록 선택
```
이름을 변경할 블록을 선택하세요: [블록 클릭!]
현재 블록 이름: CHAIR01
```

### 4단계: 새 이름 입력
```
새 블록 이름 입력 <CHAIR01>: CHAIR_TYPE-A ↵
블록 이름이 'CHAIR01'에서 'CHAIR_TYPE-A'으로 변경되었습니다.
```

### 주요 옵션
- **Enter만 입력**: 변경 취소 (기존 이름 유지)
- **같은 이름 입력**: 변경하지 않음
- **중복 이름 입력**: "이미 존재하는 블록" 경고 후 재입력

## 🎯 실전 활용 예시

### 예시 1: 블록 이름 규칙 변경
**상황:** 회사 도면 표준 변경
```
Before: TB01, CH01, DK01
BRN 실행 ↓
After: TABLE-01, CHAIR-01, DESK-01
```
**소요 시간:** 5초/개

### 예시 2: 오타 수정
```
잘못된 이름: CHIAR01 (오타!)

명령: BRN
[CHIAR01 블록 클릭]
새 이름: CHAIR01 ↵

✓ 수정 완료!
```

### 예시 3: 이름 확인
```
명령: BRN
[블록 클릭]
현재 블록 이름: UNKNOWN-BLOCK-01
새 이름 <UNKNOWN-BLOCK-01>: ↵ (Enter)

→ 이름만 확인하고 변경 안 함
```

## 💡 활용 팁

### 팁 1: 중복 이름 자동 감지
```
새 이름: TABLE01

안전하게 작업 가능!
```

### 팁 2: 기존 블록 정의 자동 삭제
변경 후 사용하지 않는 블록 정의는 PURGE로 자동 삭제됩니다.
→ 도면 파일 용량 관리!

### 팁 3: 여러 블록 연속 작업
```
BRN → 블록1 변경 → BRN → 블록2 변경...
반복 실행으로 여러 블록 빠르게 처리!
```
