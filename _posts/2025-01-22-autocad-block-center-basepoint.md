## 목적

AutoCAD 블록 작업 시 기준점이 구석에 있으면 배치, 회전, 스케일 조정이 불편합니다. BCENT 리습은 블록의 기준점을 자동으로 중심으로 이동하고, 이미 배치된 블록들의 위치를 회전과 스케일을 고려하여 자동으로 보정합니다.

## 개요

BCENT(Block Center) 리습의 주요 기능은 다음과 같습니다.

**주요 기능**
- 블록 기준점 자동 중심 이동
- 바운딩 박스 기반 중심 계산
- 배치된 블록 위치 자동 보정
- 회전 및 스케일 고려
- 여러 블록 동시 처리
- 중복 처리 방지

**기술적 특징**
- Visual LISP 기반
- 정확한 기하학적 계산
- 블록 재정의 방식
- 실시간 처리 로그

**처리 과정**
1. 선택한 블록 분석
2. 바운딩 박스로 중심점 계산
3. 블록 분해 및 객체 이동
4. 블록 재정의
5. 배치된 인스턴스 위치 보정

## 사용 방법

### 기본 사용법

```
명령: BCENT

=== 블록 기준점 중심으로 변경 ===
여러 종류의 블록을 선택할 수 있습니다.
[블록 선택]

선택된 블록 개수: 15

블록 'CHAIR-01' 처리 중...
    현재 기준점: (0.00, 0.00)
    새 기준점: (250.00, 200.00)
  ✓ 'CHAIR-01' 기준점이 중심으로 변경되었습니다.

블록 'TABLE-01' 처리 중...
    현재 기준점: (50.00, 0.00)
    새 기준점: (400.00, 300.00)
  ✓ 'TABLE-01' 기준점이 중심으로 변경되었습니다.

=== 완료 ===
총 2개의 서로 다른 블록이 처리되었습니다.
처리된 블록:
  - CHAIR-01
  - TABLE-01
```

### 처리 내용

- 같은 이름의 블록은 한 번만 처리
- 각 블록의 모든 인스턴스 자동 업데이트
- 회전 각도 보정
- X/Y 스케일 보정

## 활용 예시

### 예시 1: 가구 블록 배치

```
상황: 기준점이 구석에 있어 배치가 불편함

작업 전:
의자 클릭 → 다리 끝에 배치됨
테이블 클릭 → 모서리에 배치됨

BCENT 실행

작업 후:
의자 클릭 → 중심에 배치됨
테이블 클릭 → 중심에 배치됨
```

직관적인 배치가 가능해집니다.

### 예시 2: 심볼 회전

```
상황: 회전 시 구석을 중심으로 회전하여 위치 조정 필요

작업 전:
심볼 회전 → 구석을 중심으로 회전
위치 조정 필요

BCENT 실행

작업 후:
심볼 회전 → 중심을 축으로 회전
위치 조정 불필요
```

회전 작업이 편리해집니다.

### 예시 3: 블록 스케일 조정

```
상황: 스케일 변경 시 위치가 틀어짐

작업 전 (기준점 구석):
스케일 2배 → 한쪽으로 확대됨
위치 완전 틀어짐

BCENT 실행 (기준점 중심):
스케일 2배 → 중심 유지하며 확대
위치 유지
```

스케일 조정이 정확해집니다.

### 예시 4: 블록 라이브러리 정리

```
상황: 회사 블록 라이브러리 표준화

전체 블록 선택
BCENT 실행
5분 만에 100개 블록 정리 완료

표준화된 블록 라이브러리 구축
```

대량 블록을 빠르게 정리할 수 있습니다.

## 주의사항

**블록 정의 변경**
- 블록 기준점 변경 시 블록 정의가 수정됩니다.
- 다른 도면에서 사용하는 블록의 경우 주의가 필요합니다.

**중첩 블록**
- 블록 안에 또 블록이 있는 경우 바깥 블록만 처리됩니다.
- 안쪽 블록은 별도로 처리해야 합니다.

**작업 취소**
- U(Undo) 명령으로 전체 작업을 취소할 수 있습니다.

**AutoCAD 버전**
- AutoCAD 2000 이상, Visual LISP 필요
