## 😤 도면에서 해치 지우기가 이렇게 힘들 줄이야

도면 정리하다 보면 **해치(HATCH)를 전부 삭제**해야 할 때가 있어요.

- 파일 용량 줄이기
- 출력 속도 향상
- 간단한 선도면 만들기
- 협력업체 전달용 정리

**문제는... 해치가 너무 많다는 거죠!** 😱

**기존 방법:**
```
1. 해치 하나씩 클릭해서 삭제... (100개면 100번 클릭)
2. 또는 SELECTSIMILAR 사용 → 선택 → 삭제
3. 근데 블록 안에 숨어있는 해치는?
   → 블록 분해 → 삭제 → 다시 블록 만들기 → 💀
```

**특히 블록 내부 해치가 진짜 골치아파요!**

- 심볼 블록에 배경 해치
- 타이틀 블록에 로고 해치
- 범례 블록에 패턴 해치

이것들은 **일반 선택으로 안 잡혀요!** 😭

그래서 만들었습니다.

> "블록 속까지 숨어있는 해치도 한 방에 지울 수 없을까?"

그 답이 **BLDEL 리습**입니다! 💣

---

## ⚡ BLDEL 리습의 강력함

**일반 삭제 vs BLDEL:**

| 방법 | 일반 해치 | 블록 내부 해치 |
|------|-----------|----------------|
| 수동 삭제 | ✅ 가능 (힘듦) | ❌ 거의 불가능 |
| ERASE 명령 | ✅ 가능 (힘듦) | ❌ 불가능 |
| **BLDEL 리습** | ✅ **자동** | ✅ **자동** |

**블록 내부까지 완벽 삭제!** 🎯

## 📥 다운로드

### LISP 파일 다운로드
**파일명:** AN_BLDEL_해치삭제.lsp

👉 **[여기를 클릭하여 다운로드](https://raw.githubusercontent.com/cattowel39200/Error_404/main/AN_lisp/AN_BLDEL_%ED%95%B4%EC%B9%98%EC%82%AD%EC%A0%9C.lsp)**

### 설치 방법
1. 위 링크 클릭 → 마우스 우클릭 → "다른 이름으로 저장"
2. 파일을 `.lsp` 확장자로 저장
3. AutoCAD에서 `APPLOAD` 명령어 실행
4. 저장한 `.lsp` 파일 선택하여 로드

## 📖 사용 방법 (2가지 모드)

### 방법 1: 전체 블록 모드 (기본)

```
명령: BLDEL
모든 해치를 삭제합니다 (블록 내부 포함)...
[1-전체 블록 / 2-선택한 블록만] <1>: ↵ (Enter)

일반 해치 45개 삭제
블록 검사: 심볼-A
  - 해치 삭제됨
블록 검사: 심볼-B
  - 해치 삭제됨
...

총 67개의 해치가 삭제되었습니다.
```

**도면 전체의 모든 해치 완전 제거!** 🧹

---

### 방법 2: 선택 블록 모드

```
명령: BLDEL
[1-전체 블록 / 2-선택한 블록만] <1>: 2

해치를 삭제할 블록을 선택하세요...
[특정 블록들만 선택]
선택된 블록 종류: 3개

일반 해치 10개 삭제
블록 검사: 심볼-A
  - 해치 삭제됨
...

총 15개의 해치가 삭제되었습니다.
```

**선택적 삭제도 가능!** 👍

## 🎯 실전 활용 예시

### 예시 1: 도면 경량화

**상황:** 협력업체에 전달할 도면, 해치 때문에 용량이 30MB

```
명령: BLDEL
전체 블록 모드 (Enter)

결과:
- 해치 삭제 전: 30MB
- 해치 삭제 후: 8MB

파일 크기 73% 감소! 📉
```

**이메일 전송 가능한 크기로!** 📧

---

### 예시 2: 출력 속도 향상

**상황:** 해치가 너무 많아서 출력(Plot) 시 5분 걸림

```
Before: 출력 5분 소요
↓ BLDEL 실행
After: 출력 30초 완료

시간 절약: 90%! ⚡
```

---

### 예시 3: 선도면 작성

**상황:** 상세도를 간단한 선도면으로 만들어야 함

```
1. BLDEL로 모든 해치 삭제
2. 필요한 선만 남음
3. 선도면 완성!

작업 시간: 1분 ✨
```

---

### 예시 4: 블록 내부 해치만 정리

**상황:** 심볼 블록 안에 배경 해치가 있어서 보기 불편함

```
명령: BLDEL
모드: 2 (선택한 블록만)
[문제 있는 심볼 블록들 선택]

블록 검사: 심볼-A
  - 해치 삭제됨 (블록 안에 숨어있던 것!)

블록은 유지, 내부 해치만 제거!
```

**블록 분해 없이 내부 해치 제거!** 🎯

## 💡 꿀팁 모음

### 팁 1: 시스템 블록은 자동 제외

```
ModelSpace, PaperSpace, Xref 등은
자동으로 건너뜀!
```

**안전하게 작동!** 🛡️

### 팁 2: 작업 후 자동 재생성

```
삭제 완료 후 자동으로 REGEN 실행
→ 화면 즉시 업데이트!
```

**별도 명령 필요 없음!** 👍

### 팁 3: 블록 종류 확인 가능

```
선택 모드(2)에서:
"선택된 블록 종류: 5개"

몇 종류의 블록이 선택되었는지 확인 가능!
```

### 팁 4: 처리 과정 실시간 표시

```
블록 검사: 심볼-A
블록 검사: 심볼-B
  - 해치 삭제됨
...
```

**뭐가 진행되는지 한눈에!** 👀

## 🔥 이 리습의 장점

✅ **완벽 삭제** - 블록 내부까지 찾아서 제거
✅ **선택 가능** - 전체 or 선택 블록
✅ **안전함** - 시스템 블록 자동 제외
✅ **빠름** - 수백 개도 몇 초 안에
✅ **자동화** - REGEN까지 자동 실행

### 주의 1: 해치는 복구 불가

