## 목적

AutoCAD 도면에서 해치를 삭제할 때, 일반적인 방법으로는 블록 내부에 포함된 해치를 제거할 수 없습니다. BLDEL 리습은 도면의 모든 해치와 블록 내부의 해치까지 완벽하게 삭제하여 파일 용량을 줄이고 출력 속도를 향상시킵니다.

## 개요

BLDEL(Block Delete) 리습의 주요 기능은 다음과 같습니다.

**주요 기능**
- 도면 내 모든 해치 삭제
- 블록 내부 해치 삭제
- 전체 블록 모드
- 선택 블록 모드
- 시스템 블록 자동 제외
- 자동 재생성(REGEN)

**기술적 특징**
- Visual LISP 기반
- 블록 정의 분석 및 수정
- 안전한 블록 재정의

**처리 과정**
1. 일반 해치 삭제
2. 블록 정의 검사
3. 블록 내부 해치 검색
4. 해치 삭제 및 블록 재정의
5. 화면 재생성

**설치 방법**

2. 파일을 .lsp 확장자로 저장
3. AutoCAD에서 APPLOAD 명령 실행
4. 저장한 파일 선택 후 로드

## 사용 방법

### 방법 1: 전체 블록 모드

```
명령: BLDEL
모든 해치를 삭제합니다 (블록 내부 포함)...
[1-전체 블록 / 2-선택한 블록만] <1>: [Enter]

일반 해치 45개 삭제
블록 검사: 심볼-A
  - 해치 삭제됨
블록 검사: 심볼-B
  - 해치 삭제됨
...

총 67개의 해치가 삭제되었습니다.
```

도면 내 모든 블록의 해치를 삭제합니다.

### 방법 2: 선택 블록 모드

```
명령: BLDEL
[1-전체 블록 / 2-선택한 블록만] <1>: 2

해치를 삭제할 블록을 선택하세요...
[특정 블록 선택]
선택된 블록 종류: 3개

일반 해치 10개 삭제
블록 검사: 심볼-A
  - 해치 삭제됨
...

총 15개의 해치가 삭제되었습니다.
```

선택한 블록의 해치만 삭제합니다.

## 활용 예시

### 예시 1: 도면 경량화

```
상황: 협력업체 전달용 도면, 해치로 인한 대용량 파일

작업 전: 30MB
BLDEL 실행
작업 후: 8MB

파일 크기 73% 감소
```

이메일 전송 가능한 크기로 축소됩니다.

### 예시 2: 출력 속도 향상

```
상황: 해치가 많아 출력이 느림

작업 전: 출력 5분 소요
BLDEL 실행
작업 후: 출력 30초 완료

시간 절약 90%
```

출력 작업이 신속하게 진행됩니다.

### 예시 3: 선도면 작성

```
상황: 상세도를 간단한 선도면으로 변환

BLDEL로 모든 해치 삭제
필요한 선만 남음
선도면 완성

작업 시간 1분
```

깔끔한 선도면을 빠르게 작성할 수 있습니다.

### 예시 4: 블록 내부 해치 정리

```
상황: 심볼 블록 배경 해치가 보기 불편함

명령: BLDEL
모드: 2 (선택한 블록만)
대상: [문제 블록 선택]

블록은 유지, 내부 해치만 제거
```

블록을 분해하지 않고 내부 해치만 제거할 수 있습니다.

## 주의사항

**백업 권장**
- 삭제된 해치는 복구할 수 없습니다.
- 중요한 해치가 있는 경우 백업을 권장합니다.
- U(Undo) 명령으로 전체 작업 취소 가능

**블록 정의 변경**
- 블록 내부 해치 삭제 시 블록 정의가 수정됩니다.
- 다른 도면에서 사용하는 블록의 경우 주의가 필요합니다.

**시스템 블록**
- ModelSpace, PaperSpace, Xref 등은 자동 제외됩니다.
- 안전하게 작동합니다.

**AutoCAD 버전**
- AutoCAD 2000 이상, Visual LISP 필요
